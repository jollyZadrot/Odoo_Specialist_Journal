Specialist Journal (Журнал спеціаліста)
Модуль для Odoo 19, призначений для ведення детальної історії взаємодії з клієнтами. Основна фішка — швидке заповнення нотаток за допомогою шаблонів та повна ізоляція даних між спеціалістами.

Логіка роботи
Програма базується на трьох принципах:

Конфіденційність: Спеціаліст бачить у базі тільки ті записи, які створив він сам. Адміністратор бачить повну картину по всіх працівниках.

Швидкість: Замість того, щоб щоразу писати однотипні тексти, спеціаліст в один клік додає готові блоки (шаблони), які потім редагує.

Цілісність: Спеціаліст не має права видаляти записи. Це гарантує, що історія клієнта не буде випадково або навмисно стерта.

Основні функції
Вкладка в Контактах: Журнал інтегрований прямо в картку клієнта.

Розумні шаблони: Система Many2many дозволяє вибрати кілька шаблонів поспіль, і їхній текст буде додаватися один за одним у поле нотаток.

Автоматичне скидання: Після вибору шаблону поле вибору очищується автоматично — це дозволяє зручно вибирати наступний шаблон.

PDF-звіти: Можливість вивантажити вибрані записи в офіційний PDF-документ з логотипом вашої компанії.

Поділ ролей: Чітке розмежування прав доступу через групи користувачів.

Посібник користувача
1. Початкове налаштування (для Адміністратора)
Створення шаблонів: Перейдіть у Контакти → Конфігурація → Шаблони журналів. Створіть типи записів (наприклад: "Первинний огляд", "Рекомендації", "План лікування").

Налаштування прав: Зайдіть у Налаштування → Користувачі. У картці кожного працівника в полі "Журнал спеціаліста" виберіть відповідну роль:

Спеціаліст: Бачить тільки своє, не може видаляти.

Адміністратор: Бачить все, керує шаблонами.

2. Ведення журналу (для Спеціаліста)
Відкрийте картку потрібного клієнта в модулі Контакти.

Перейдіть на вкладку "Журнал записів".

Натисніть "Додати рядок".

У колонці "Шаблон" (позначена блискавкою ⚡) виберіть один або декілька шаблонів. Текст автоматично з'явиться в полі поруч.

Відредагуйте текст під конкретну ситуацію та збережіть картку клієнта.

3. Друк звіту
Перейдіть у загальний список записів (меню Контакти → Журнал).

Виберіть потрібні записи галочками.

Натисніть кнопку "Друк" (Print) у верхній частині екрана.

Виберіть "Журнал (PDF)". Система згенерує файл, де будуть лише ті записи, до яких у вас є доступ.
